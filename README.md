# MNIST From Scratch (Neural Network Implementation)

This project demonstrates how to build and train a neural network from scratch on the MNIST dataset without using high-level deep learning frameworks.

## 🔍 Overview

The goal of this project is to understand the fundamentals of deep learning by implementing:

- Forward propagation  
- Backpropagation  
- Gradient descent optimization  
- Neural network training loop  

## 🧠 Model

- Fully connected neural network  
- Trained on MNIST handwritten digit dataset  
- Implemented using NumPy  

## ⚙️ Features

- No TensorFlow / PyTorch high-level APIs  
- Manual implementation of training process  
- Educational and research-oriented  

## 📊 Results

The model learns to classify handwritten digits (0–9) with reasonable accuracy using a custom implementation.

## 🚀 Usage

Run the notebook:

```bash
jupyter notebook notebook/mnist_from_scratch.ipynb
```
